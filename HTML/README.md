# HTML cheatsheet

## Characters

&amp;nbsp; – space (non-breaking)  
&amp;quot; - add quotation mark (“)  
&amp;lt; - less than symbol (&lt;)  
&amp;gt; - greater than symbol (&gt;)  
&amp;amp; - the ‘&amp;’ or ampersand symbol  
&amp;copy; - copyright symbol  
&amp;trade; - trademark symbol
